# Hello World — Why This Blog Exists

*Januari 5, 2026*

Every developer's first program prints "Hello, World!" — a small declaration of existence in a vast digital landscape. This blog is my version of that.

I've been writing code professionally for years, but I've always felt that the most interesting interactions happen at the edges — where technology meets ideas, where a piece of code reveals something about how we think, where debugging teaches patience more than any meditation app ever could.

This space is for conversations about those interactions. Expect posts about code and AI development, and the occasional detour into questions like: *What does it mean to build something that outlasts you?*

Thanks for being here. Let's see where this goes.
